便捷指令集

/ /screenshot：微信一键实时截取本地 IDE 画面
/ /tasks：查看本地运行中的后台任务列表
/ /kill <ID>：远程终止指定的卡死任务或进程
/ /status：获取本地系统健康状况 (CPU/内存/磁盘)
/ /sleep：远程使本地电脑进入睡眠状态
/ /lock：远程锁定本地电脑屏幕
/ /reset：重置对话上下文，清空 Agent 记忆
/ /awake：开启/关闭本地挂机防掉线，防止系统进入睡眠状态
