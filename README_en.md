# WeChat with Agent 🚀 (Native Agent Bridge)

<p align="center">
  <strong>Turn your WeChat into a universal remote for your AI Agents!</strong>
</p>

This project allows you to chat directly with locally running **AI Agents (like Antigravity IDE, Claude Code, etc.)** using WeChat. It supports text, images, files, and even voice-based multimodal interaction!

Following a deep architectural refactoring, we have completely abandoned traditional background polling daemons in favor of a **Native Agent Loop** tightly integrated with the Agent engines. This isn't just a chatbot; it's a **full-stack AI developer** hanging out in your WeChat contacts.

---

## 🌟 Highlights

| | |
|---|---|
| **Scan and go** | No complex server architecture, no tedious signups. Scan the QR code on your PC, and you're ready in a minute. All data flows locally—100% privacy guaranteed. |
| **Clean messages** | Only key information is pushed to WeChat—progress and final results. Intermediate noise like terminal outputs, bug fixes, and tool calls are automatically intercepted. |
| **"Typing..." indicator** | When the agent is executing complex tasks that take minutes, a cross-process background daemon maintains the "typing..." animation at the top of WeChat, giving you peace of mind that it's working. |
| **Two-way files** | Supports voice-to-text and multimodal audio recognition. Send screenshots, PDFs, or bugs for it to analyze. Reports and code generated by the agent can be sent directly back to WeChat. |
| **Dynamic Brain Transfer** | WeChat becomes a remote roaming seamlessly across your codebase universes! Trigger the listener in ANY open terminal or workspace, and that window instantly takes over WeChat control for that specific project. |
| **Race-condition Mutex** | Built-in physical Mutex file locks. Prevents "multiple brains" (different workspaces) from simultaneously fighting for your WeChat messages. |

## 🛠️ Prerequisites

1. Node.js (v18+) and npm installed locally.
2. A running Agent platform that supports terminal commands or standard AI Skills (e.g., Antigravity IDE, Claude Code).

---

## 🚀 Quick Start Guide

### Step 1: Initialize and Login

First, install dependencies and run the setup script in the project directory.
```bash
npm install
npm run setup
```
A QR code will pop up on your screen. Scan it with WeChat. Upon success, your credentials will be stored locally in the `.data` directory.

### Step 2: Launch the Service (Choose one)

**Option A: Pure Terminal Native Launch**
You can start the listener in any workspace you want the agent to control by directly executing:
```bash
# Ensure you are in your target project workspace and pass the absolute path to the script
node /path/to/wechat-with-antigravity/dist/agent-loop/wait-message.js
```

**Option B: Elegant AI Skill Integration (Highly Recommended)**
If your Agent platform (like Claude Code or advanced IDEs) supports the standard `SKILL.md` skill extension mechanism, we strongly recommend integrating it as a global Skill:
1. Locate the `SKILL.md` file in the root of this project.
2. Place it into your Agent platform's prescribed skills directory (e.g., `~/.claude/skills/wechat-listener/` or your IDE's plugin folder).
3. Trigger the skill via its invoke command. Your current workspace will instantly transform into the global WeChat listening base station!

## 🌙 Awake Mode

To achieve 24/7 round-the-clock message responsiveness, the underlying system bundles a script that calls Windows APIs to prevent sleep. By running the following command, your screen can turn off, but the system will never enter deep sleep or cut the network:
```bash
npm run awake
```

## 🧹 Memory Reset (/reset)

LLM memory can bloat over long conversations. When you want the agent to drop the current context and start fresh with a blank slate, simply send this command in WeChat:
👉 **"/reset"** or **"forget the previous tasks and start a new one"**

Upon receiving the command, the agent will actively intercept the request and clear the current task stream, resetting its brain to a blank page!
